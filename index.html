"Desligar o codigo"

  • Para "desligar" o código basta digitar o comando // quando for apenas uma linha
    de código, para desligar mais de uma linha, usa-se o comando /* e fecha com */.

  • Este "Desligar o código" é basicamente escrever um código mas fazer com que
    ele não funcione por enquanto.
    (Tem como usar o // também depois de uma linha de código para colocar um comentario sobre
    oque ela faz ou sobre qualquer coisa. )

    Data type

    Oque é um data type? Bom, um data type é a classificação das variáveis.
    Neste curso usaremos 3 como os principais, que são:

  • Number = todos os números
  • String = conjunto de caracteres (números, letras, símbolos todos juntos)
  • Boolean = True or False
